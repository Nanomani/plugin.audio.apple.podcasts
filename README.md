## Apple Audio Podcasts

This add-on lets you browse and search for thousands of free audio Podcasts.  
You can search for specific podcasts by name.  
It is also possible to manage a "My Podcasts" list via context-menu.

[Latest release](https://github.com/Nanomani/plugin.video.apple.podcasts/releases)
